#+title: 1081. Smallest Subsequence of Distinct Characters

* Problem Description
Given a string s, return the of s that contains all the distinct characters of s exactly once.

* Examples 

Example 1:

Input: s = "bcabc"
Output: "abc"

Example 2:

Input: s = "cbacdcbc"
Output: "acdb"

 

* Constraints:

    1 <= s.length <= 1000
    s consists of lowercase English letters.
